The purpose of P2P build is to
* Deploy to production

### Deploy Application

Deploy to the following namespace

```<app_name>-prod```

Or if you have multiple apps in the same tenancy

```<tenant>-<app-name>-prod```

Variables set by the P2P that you can use:
* REGISTRY
* VERSION
